# Fileszzzzzzz

RIA1 - Lab


## Setting up dev

### Prerequisites
-  npm : >= '8.5.4',
-  node: >= '17.7.1',

### Clone repository

```shell
git clone git@github.com:melohan/Fileszzzzzzz.git
cd Fileszzzzzzz
```

### Run test

Run all tests
```shell
npm run test
```

Run a single test
```shell
 npm run test [className.test.js]
```
